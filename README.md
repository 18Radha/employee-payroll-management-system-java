# Employee Payroll Management System (Java)

## 📌 Project Overview

The **Employee Payroll Management System** is a console-based Java application designed to manage employee payroll records and automatically calculate salaries based on predefined business logic.

This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts and core Java features to simulate a basic payroll system used in organizations.

The application allows users to add employees, calculate salaries, and view payroll details through a menu-driven interface.

## 💰 Salary Calculation Logic

| Component    | Formula                   |
| ------------ | ------------------------- |
| HRA          | 20% of Basic Salary       |
| DA           | 10% of Basic Salary       |
| Gross Salary | Basic + HRA + DA          |
| Deductions   | 8% of Gross Salary        |
| Net Salary   | Gross Salary − Deductions |
